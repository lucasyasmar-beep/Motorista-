# Motorista – Meta & Relatórios (PWA)

## Como usar pelo celular (Android)
1. Envie a pasta do app para algum lugar que tenha link (ex: GitHub Pages, Netlify, Vercel, ou hospedagem simples).
2. Abra o link no Chrome.
3. Menu ⋮ -> **Adicionar à tela inicial** (instala como app).
4. Funciona offline após o primeiro acesso.

## Dica rápida de hospedagem (sem computador)
- Você pode usar serviços que aceitam upload pelo celular (ex: Netlify Drop), enviando o ZIP e publicando.

## O que o app faz
- Ganho do dia (total)
- Gastos manuais por categoria
- Meta diária editável
- Relatórios: diário, semanal e mensal
- Exportar CSV e salvar PDF via impressão

Dados ficam somente no seu aparelho (localStorage).
